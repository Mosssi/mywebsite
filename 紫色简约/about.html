About This Project

This site is a prototype for the assignment. It demonstrates: semantic HTML structure, a sticky header with a flex‑based menu, responsive images, a background image on the Home hero, and a mailto contact form with required fields.

  • External CSS files (typography.css, main.css).
  • Flexbox (nav, About split) and Grid (card layout, Gallery).
  • Media query at 768px for tablet/desktop.
  • Active menu state on each page.
  • Consistent BEM‑ish naming: .nav__link--active, .card__title.

© 2025 MiniSite. Built for the multi‑page website assignment. Images are local SVG placeholders.
